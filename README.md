video-server
============

Setup
-----

```
npm update
```

Download media files (e.g. big_buck_bunny.webm, .mp4, .ogv) and put into folder 'media'


Getting Started
---------------

```
node app.js
```

Open browser and goto [http://localhost:3000/media/big_buck_bunny.webm] (http://localhost:3000/media/big_buck_bunny.webm)
